# The Caverns of Freitag

[![The Caverns of Freitag](http://img.youtube.com/vi/GxaAfNZJURE/0.jpg)](https://www.youtube.com/watch?v=GxaAfNZJURE)

## SIDELOAD
Releases にある freitag-xxx.d88 ファイルをダウンロード、各種 X1 環境で動かしてください。

## 操作
- `T` `Y` `U`, &ensp;&ensp;&ensp;`7` `8` `9`<br>
`G` `_` `J`, &ensp;&ensp;&ensp;`4` `_` `6` （テンキー）<br>
`B` `N` `M`, &ensp;&ensp;&ensp;`1` `2` `3`<br>
8 方向に移動<br>
`SHIFT` あるいは `CTRL` を押しながら方向を入力すると攻撃
- `H`, `5`<br>パス
- `SPACE`<br>武器（剣、弓矢）の切り替え
- `ESC`<br>鳥に変身（攻撃できないが 2 倍速で移動）
- `A`<br>INN で矢を購入
- `P`<br>INN で最大 HP を購入
- `1`, `2`, `3`<br>マップ表示の切り替え
- `O`<br>メッセージの ON/OFF
- `:`<br>自分を中心にした表示に更新
